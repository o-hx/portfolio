menü App

Project for 50.001 - Intro to Information Systems & Programming. menü is a menu builder which allows vendors to quickly list their items and receive cashless payment.

App features:

• Account registration
• QR code generator (for vendors)
• QR code scanner (for users) to access menu
• Implementation of OCBC PayNow API as payment system
• Intuitive menu design

My contributions to the project:

• Checkout page design
• Pop-up window in main menu - design and development
• QR scanner

Made with Android Studio.
